Robot.txt and site index

Make sure robots.txt is enabled if you want search engines to crawl your site. Also upload your site index to google search console so Google indexes your site faster.

Title

11 words or less with keyword Make sure keyword is not at the beginning of the title.

Optimize Headings for SEO:

# Main Heading (title) H1 (is this title in Hugo?)

## H2 first main section

### H3 subheading under first H2

## H2 second main section

### H3 under second main section

### H3 same

Article length

Must be at least 300 words

Keywords and key-phrases

Something someone is searching for on Google to learn something about your product

Keyword or key-phrase needs to be in:

  • title
  • in a heading
  • in text

Don’t stuff keywords. One keyword per above category is fine. Use natural writing. (Conversational tone)

Use in first paragraph.

Don’t use the same keyword on multiple articles.

Links

Include both types to help with SEO. Make links useful to the reader.

Internal link

Link to another page on your site.

External link

Link to another site.

Meta Data

Description under title that you see in google. (Is this the summary or description in Hugo?)

Between 11 and 21 Words
